Requirements

You will need the following sofwares to customize this script.

  1. Xampp or Ampps or Wamp or Mamp or Lamp , you can use any of your local development environment to make changes in it
  2. Required php 7.2 or higher
  3. Code Editing Software (eg: VS code, Sublime Text or Notepad)
  4. Web Browser for testing (eg: Google Chrome or Mozilla Firefox)
  5. FTP Tool to upload files to Server (eg: FileZilla)

Be careful while editing the script. If not edited properly, the design layout may break completely.
No support is provided for faulty customization.
